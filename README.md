# Rede Alerta Frontend

Frontend em React + Vite + Tailwind para a homepage do projeto Rede Alerta.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy na Vercel

1. Envie esta pasta para um repositório GitHub
2. Importe o repositório na Vercel
3. Framework preset: Vite
4. Build command: `npm run build`
5. Output directory: `dist`
